1. Clone the Backend Repository:
   Make sure to clone and run the backend before starting the frontend.

2. Create .env File:
   -> In the project root, create a .env file.
   -> VITE_APP_BACKEND_URL=http://localhost:5249.
   -> Make sure this matches your backend URL and port.

3. Install Dependencies:
   npm install

4. Run the Frontend
   npm run dev
